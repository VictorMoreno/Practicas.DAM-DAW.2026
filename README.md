# 📚 Prácticas DAM-DAW 2026

Repositorio de documentación para las sesiones de prácticas de los Ciclos Formativos de Grado Superior:

- **DAM** - Desarrollo de Aplicaciones Multiplataforma
- **DAW** - Desarrollo de Aplicaciones Web

## 📋 Descripción

Este repositorio contiene toda la documentación, recursos y materiales relacionados con las reuniones y sesiones de prácticas para los estudiantes de DAM y DAW del año 2026. Aquí encontrarás apuntes, enlaces a grabaciones, ejemplos de código y recursos adicionales de cada sesión.

## 🗂️ Estructura del Repositorio

Cada clase está organizada en su propia carpeta siguiendo el formato:

```
Clase XX - Nombre del Tema/
├── Apuntes.pptx
├── Ejemplos/
└── Recursos/
```

### Contenido de cada carpeta

- **Apuntes.pptx**: Resumen y presentación de la clase
- **Ejemplos/**: Código y ejemplos prácticos
- **Recursos/**: Material complementario (PDFs, enlaces, etc.)

## 🚀 Cómo Usar Este Repositorio

1. **Clonar el repositorio**:

   ```bash
   git clone https://github.com/tu-usuario/Practicas.DAM-DAW.2026.git
   ```

2. **Navegar por las clases**: Cada sesión está en su propia carpeta numerada

3. **Acceder a las grabaciones**: Consulta el archivo `Enlace de la grabación.txt` en cada carpeta

4. **Revisar recursos**: Explora los ejemplos y materiales adicionales de cada tema
